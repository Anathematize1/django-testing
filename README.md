# Django Testing
Набор автоматических тестов для Django-приложений, покрывающий бизнес-логику, маршруты и права доступа пользователей.


## 📌 Описание проекта:
  Django Testing — это проект, направленный на проверку корректности работы веб-приложений на Django с помощью автоматических тестов.
  Основное внимание уделено тестированию моделей, представлений, маршрутов и системы прав доступа, что позволяет обеспечить стабильность приложения и корректность его поведения.


## 🚀 Функциональность:

  ### 🧪 Тестирование моделей:
  - проверка создания объектов  
  - проверка корректности полей  
  - валидация данных  
  
  
  ### 🌐 Тестирование маршрутов (URLs):
  - проверка доступности страниц  
  - проверка HTTP-статусов  
  - проверка редиректов  
  
  
  ### 👤 Тестирование прав доступа:
  - проверка доступа анонимных пользователей  
  - проверка доступа авторизованных пользователей  
  - ограничение доступа к чужим объектам  
  
  
  ### 📝 Тестирование CRUD-логики:
  - создание объектов  
  - редактирование  
  - удаление  
  - проверка изменений в базе данных  


## 🛠 Стек технологий:
- Python  
- Django  
- pytest  
- Django Test Client  


## ⚙️ Запуск тестов:
1. Установить зависимости:
```bash
pip install -r requirements.txt
```
2.Запустить тесты:
```bash
pytest
```


## 📌 Особенности проекта:
- реализовано тестирование бизнес-логики приложения
- проверяется корректность работы маршрутов и представлений
- реализовано тестирование прав доступа пользователей
